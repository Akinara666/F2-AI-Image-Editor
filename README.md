**Графический редактор с генеративным ИИ и интеллектуальной оптимизацией промптов**

## О проекте
Этот проект посвящён разработке редактора изображений, который использует языковую модель для улучшения пользовательских текстовых запросов (prompt engineering).  
Цель — сделать генерацию изображений более доступной, понятной и управляемой.

## Основные возможности
- Генерация изображений по простым запросам;
- Автоматическое улучшение промпта языковой моделью;
- Удобный интерфейс для визуальной работы.
